# Mahatma Gandhi Institute of Rural Energy and Development (MGIRED) 🏛️🌱

> Under the Department of Rural Development and Panchayat Raj, Government of Karnataka.

An ultra-premium, high-fidelity, responsive web portal built for **MGIRED**, designed to empower rural communities through sustainable energy, innovative technology, and state-of-the-art skill development initiatives.

---

## ✨ Features & Capabilities

- **🏛️ Symmetrical Government Branding**: Incorporates authentic Karnataka State Emblem and MGIRED logos with luxurious, clean typographic hierarchy.
- **🌓 Adaptive Theme Engine**: Starts in a beautiful, premium **Dark Theme** by default (session-persisted) with a seamless, polished toggle to a tailored high-contrast **Light Theme**.
- **📅 Model Heritage Village Portal**: Includes interactive visual masonry grids showcasing Karnataka's rich cultural, agricultural, and architectural heritage.
- **🎓 Training & Internship Management**: Fully detailed syllabus structures, eligibility tracking, and custom application guides for state-level green-energy certifications.
- **🌾 Centralized Government Schemes**: Curated database of state and national energy subsidies, application links, and execution flow charts.
- **📱 Responsive Luxury Experience**: Fluidly optimized for desktop, tablet, and mobile displays down to 320px width, utilizing modern viewport clamps.

---

## 🛠️ Technology Stack

- **Framework**: React 18 with Vite
- **Styling**: TailwindCSS & Vanilla CSS Transitions
- **Animations**: Custom scroll-reveal hooks and performance-tuned micro-interactions
- **Localization**: i18next supporting both **English** and **ಕನ್ನಡ (Kannada)**
- **Icons**: Lucide React

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/KUKI-Boi/MGIRED.git
cd MGIRED
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```
